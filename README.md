**testing lang po**.

## Features

- **System Maintenance** – Update and upgrade your system with a single option.  
- **Downloader Installation** – Automatically installs `yt-dlp` if missing.  
- **Single Video Download** – Download any video from supported sites with a choice of resolution or audio only.  
- **Batch Downloads** – Download multiple videos by reading a list of URLs from a text file.  
- **Folder Organization** – Downloads are organized by **date** and **site domain** automatically.  
- **Logging** – Tracks updates and downloads in `~/video_bot.log`.  
- **Colorful CLI Interface** – Easy-to-read prompts and status messages.
